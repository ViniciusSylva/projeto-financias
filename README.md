# 💰 Financial Control App

Aplicação web simples para controle de finanças pessoais.

O objetivo do projeto é permitir que o usuário registre seus gastos, visualize o total e acompanhe quais valores já foram pagos.

## 📊 Funcionalidades

- adicionar novos gastos
- visualizar lista de despesas
- marcar despesas como pagas
- visualizar total de gastos
- visualizar total ainda a pagar

## 🧰 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 🖥️ Interface

A aplicação possui uma interface simples com tema escuro para facilitar a visualização das informações financeiras.

## ⚙️ Como executar o projeto

1. Clone o repositório

2. Abra a pasta do projeto

3. Execute o arquivo: em seu navegador.

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- manipulação do DOM
- lógica de programação em JavaScript
- organização de dados
- construção de interfaces simples

## 📌 Possíveis melhorias
